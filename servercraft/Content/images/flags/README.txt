Flag Images Required:

1. gb.png - United Kingdom flag (16x16 pixels)
2. ro.png - Romania flag (16x16 pixels)
3. ru.png - Russia flag (16x16 pixels)

You can download these flag images from:
- https://www.countryflags.io/gb/flat/64.png (rename to gb.png)
- https://www.countryflags.io/ro/flat/64.png (rename to ro.png)
- https://www.countryflags.io/ru/flat/64.png (rename to ru.png)

After downloading, resize them to 16x16 pixels and place them in this directory.

Alternatively, you can use any other source for flag images, such as:
- https://flagicons.lipis.dev/
- https://github.com/lipis/flag-icons
- https://github.com/hampusborgos/country-flags

Make sure the images are:
1. In PNG format
2. Named exactly as specified (gb.png, ro.png, ru.png)
3. Resized to 16x16 pixels for consistent display
4. Placed directly in this directory
